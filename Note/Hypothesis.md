สมมติฐานเบื้องหลังกลยุทธ์ Trend Following คือ
1.  ราคาเคลื่อนไหวเป็นแนวโน้มและมีความต่อเนื่อง
	โดยราคาเมื่อเคลื่อนไหวแล้ว ราคาก็มักจะมีแนวโน้มที่ดำเนินต่อไป แทนที่จะเกิดการกลับตัวในทันที เนื่องจากพฤติกรรมของตลาดนั้นซึบซับข่าวสารได้ช้ากว่าความเป็นจริง และเมื่อรับรู้แล้วนักลงทุนก็มักจะมีพฤติกรรมแห่ซื้อตามราคาที่ขึ้นไปแล้ว กลยุทธ์จึงใช้ข้อมูลเส้นทางราคาในอดีตมากำหนดสัญญาณเข้าซื้อเมื่อเป็นขาขึ้น ชอร์ตเมื่อเป็นขาลง และไม่ถือสถานะเมื่อสัญญาณไม่ชัด
2. ผลตอบแทนแบบไม่สมมาตร
	เนื่องจากระบบเน้นการเกาะไปกับแนวโน้ม กลยุทธ์จึงยอมรับการขาดทุนจำนวนเล็กน้อยหลายครั้ง เพื่อแลกกับการจับกำไรก้อนใหญ่ กล่าวคือจะลดหรือปิดสถานะเมื่อสัญญาณกลับทิศ อ่อนแรง หรือแตะเพดานความเสี่ยง แต่จะถือสถานะไว้ให้นานที่สุดเมื่อเกิดการเคลื่อนไหวครั้งใหญ่
3. ความเป็นสถาปัตยกรรมของระบบองค์รวม
	สมมติฐานนี้จะใช้ได้จริงก็ต่อเมื่อ
	* ระยะเวลาของเทรนด์ เกิดนานพอซึ่งสำคัญกว่าตัวสัญญาณมาก
	* โอกาสที่กว้างพอ ต้องกระจายการลงทุนให้หลากหลายตลาดเพื่อให้มีโอกาสมากพอจะเจอเทรนด์ใหญ่ ไม่ว่าจะเป็นขาขึ้นหรือลง
	* การจัดการความเสี่ยง เช่น การใช้ Volatility เพื่อปรับขนาดสัญญา
	* การควบคุมต้นทุน ต้นทุนการเทรดที่สูงจากการหมุนรอบจะทำลายกำไรของระบบ

การทดสอบสมมติฐาน
1. ถ้าแนวโน้มมีลักษณะดำเนินต่อไปจริง baseline 60-day directional count ต้องมี positive convex payoff หลังหัก cost
2. multi-lookback t-stat ต้องให้ risk-adjusted return ดีกว่า baseline หรือ turnover/cost profile ดีกว่า
3. inverse vol sizing ต้องลด concentration ของ risk เมื่อเทียบกับ equal notional
4. multi-asset universe ต้องลด dependency ต่อ trend ของตลาดเดียว
